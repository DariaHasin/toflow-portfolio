# TOFlow Portfolio

Public portfolio version of TOFlow: Multi-Plane TOF-MRA Reconstruction.

This is a simplified version designed for CV and public viewing, showcasing results and high-level approach without revealing detailed methodology.

## Development

```bash
npm install
npm run dev
```

## Deployment

This site is automatically deployed to GitHub Pages when changes are pushed to the main branch.

## About

TOFlow is an advanced reconstruction pipeline for Time-of-Flight Magnetic Resonance Angiography (TOF-MRA) that enhances both spatial resolution and directional flow representation.
